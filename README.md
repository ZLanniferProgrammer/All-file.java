# All-file.java
# Quest System Upgrade

โปรเจกต์นี้เป็นส่วนหนึ่งของรายวิชา Java Programming II  
พัฒนาต่อยอดจากระบบ Quest เดิม โดยเพิ่มฟีเจอร์ใหม่ตามที่กำหนด

## ฟีเจอร์ที่เพิ่ม
1. เพิ่มระบบจัดอันดับคะแนน (Ranking System)  
   ระบบจะเก็บคะแนนจากการเล่นรอบล่าสุดไว้ 5 ครั้ง โดยใช้ ArrayList
2. เพิ่มการใช้งาน Interface (Rewardable)  
   เพื่อกำหนดโครงสร้างการให้รางวัลของ Quest แต่ละประเภท ทำให้โค้ดเป็นระเบียบและขยายได้ง่าย

## วิธีทดสอบระบบ
1. รันโปรแกรมผ่านไฟล์ Main.java
2. เลือกเมนู List เพื่อแสดงรายการ Quest
3. เลือกเมนู Do quest และกรอก quest id
4. ทดสอบกรณีทำ Quest สำเร็จ คะแนนจะเพิ่ม
5. ทดสอบกรณีเงื่อนไขไม่ผ่าน ระบบจะแจ้งว่าไม่สามารถทำได้
6. เลือกเมนู Stats เพื่อดูคะแนน completed / failed และ ranking
7. ทดสอบกรอกเมนูผิด ระบบไม่ล้ม
8. เลือก Exit เพื่อจบโปรแกรมและแสดง signature
